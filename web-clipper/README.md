# Ktree 剪藏(Web Clipper)

把网页正文或选中内容一键剪藏到 Ktree 知识库:扩展在页面里提取正文(顶部附标题与来源链接),
以 `<标题>.html` 上传到 Ktree 既有的 `POST /api/upload?...&convert=md`,服务端用 turndown
转成 Markdown 并入库。**无需任何服务端改动**,复用现有上传与转换管线。

## 安装(Chrome / Edge,MV3)

1. 打开 `chrome://extensions/`(Edge 为 `edge://extensions/`)
2. 右上角开启「开发者模式」
3. 点「加载已解压的扩展程序」,选择本目录 `web-clipper/`
4. 固定扩展图标到工具栏

## 使用

1. 点扩展图标打开弹窗
2. 填 **Ktree 地址**(默认 `http://127.0.0.1:8080`,局域网填对应主机:端口),点 ⟳ 拉取知识库列表
3. 选 **目标知识库** 和 **子目录**(落在上传区 `src/upload/<子目录>/`,默认 `clips`)
4. 点 **剪藏整页**(自动取 `article`/`main`/`body` 正文)或 **剪藏选区**(先在页面选中文字)

配置(地址 / 知识库 / 子目录)会记在浏览器本地,下次自动带出。

## 说明与边界

- **正文提取**是轻量启发式:优先 `article` → `main` → `body`,并移除 `script/style/nav/header/footer/aside/form` 等噪声节点;复杂页面可能仍有残留,选区剪藏更精确。
- 图片 / 链接的相对地址会被绝对化,转 md 后仍可解析;图片仍指向原站远程地址。
- `chrome://`、扩展商店等受限页面无法注入,会提示「注入失败」。
- 剪藏结果落在上传区,可被搜索 / 语义检索;来源 URL 以正文顶部链接行保留。
